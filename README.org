This branch ~executable-jar~ stores demo for creating an executable jar using ~jar~ command. Do not delete this branch or it may be hard to return here!

To build the executable jar:

1. Compile the source files. Under the project root directory run:
   
   : cd src
   : javac com/josephtesfaye/javastarter/*.java
   
2. Create the jar using the ~jar~ command:

   : jar cfe java-starter.jar com.josephtesfaye.javastarter.Main com/josephtesfaye/javastarter/*.class

3. Run the jar:

   : java -jar java-starter.jar

4. To update the jar, recompile the source files and then update its contents using:
   
   : jar uf java-starter.jar com/josephtesfaye/javastarter/Main.class
